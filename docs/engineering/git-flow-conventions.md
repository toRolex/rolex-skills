## 功能

Git Flow 分支管理与提交规范指南。覆盖完整 Git Flow 模型（develop/feature/release/hotfix）、分支命名规范、commit message 格式、PR 提交流程和最佳实践。

## 何时使用

敲 `/git-flow-conventions` 或 agent 自动调用。

进行分支操作、合并代码、提交 PR/MR、规划发版、修复线上 Bug、或询问 Git 协作规范时。

## 在流程中的位置

独立参考工具。不确定时问 `/ask-rolex`。
